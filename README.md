🎵 Tunes Player App


A Flutter-based music player application that allows users to play and control audio tracks with a smooth and interactive user experience.


📌 Overview 


The Tunes Player App is a mobile application built باستخدام Flutter لعرض وتشغيل الملفات الصوتية (Music Tracks) مع واجهة مستخدم بسيطة وسلسة.


This project demonstrates your ability to work with:


🎧 Audio playback in Flutter

🎛️ Media controls (play, pause, etc.)

🎨 UI design for media applications


Audio playback in Flutter is commonly handled using packages like audioplayers, which support playing audio across multiple platforms.


✨ Features

▶️ Play audio tracks

⏸️ Pause and resume music

🎚️ Basic playback controls

🎨 Clean and modern UI

⚡ Smooth performance

🛠️ Tech Stack

Flutter

Dart

Audio Player Package (e.g., audioplayers)

📂 Project Structure

lib/
├── models/        # Data models (if used)
├── screens/       # UI screens
├── widgets/       # Reusable components
└── main.dart      # Entry point

🚀 Getting Started

Prerequisites

Flutter SDK

VS Code / Android Studio

Emulator or physical device

Installation

git clone https://github.com/AliSayed15/tunes_player_app.git

cd tunes_player_app

flutter pub get

flutter run

🎯 How It Works

The app loads audio files (local or online)

Audio player initializes

User controls playback (play / pause)

UI updates based on player state
